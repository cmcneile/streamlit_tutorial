#  Installing and running streamlit

Unfortunately streamlit doesn't work well with jupyter notebooks.

To install streamlit open up the anconda prompt

pip install streamlit

![Installing streamlit](https://github.com/cmcneile/streamlit_tutorial/blob/main/pictures/install_streamlit.png)


To test the installation run

streamlit hello

in the annconda prompt window.

This will bring up a browser window.


![Running streamlit](https://raw.githubusercontent.com/cmcneile/streamlit_tutorial/refs/heads/main/pictures/basicStreamLit.png)
